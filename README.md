# NewsPortal - Next.js 15 News Website

A modern, responsive news portal built with Next.js 15, React 19, and TailwindCSS. Features dark mode, SEO optimization, and a clean, original design.

## Features

- **Next.js 15** with App Router
- **React 19** for latest features
- **TailwindCSS** for styling
- **Dark Mode** with system preference detection
- **Framer Motion** animations
- **SEO Optimized** with metadata API
- **Responsive Design** (mobile-first)
- **Static Site Generation** for best performance
- **Image Optimization** with Next.js Image
- **Accessibility** (WCAG compliant)
- **12 Realistic Indian News Articles**

## Project Structure

```
new-assignment/

│   ├── layout.js          # Root layout

│   ├── globals.css        # Global styles
│   └── news/[slug]/       # Dynamic article pages
├── components/            # Reusable components
├── data/                  # News data (JSON)
```

## Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Usage

1. Open http://localhost:3000 in your browser
2. Browse news articles by category
3. Toggle dark mode with the theme button
4. Click on any article to read the full story
5. Use category filters to find specific news

## Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## Design System

- **Colors**: Red/Orange theme (original, not copied)
- **Typography**: Inter font family
- **Spacing**: Tailwind's 4px base unit
- **Animations**: Framer Motion for smooth transitions
